# Wikipedia Viewer

**Build a Wikipedia Viewer** is a web application that allows users to search for Wikipedia entries and explore random articles using the Wikipedia MediaWiki API. Inspired by the freeCodeCamp example, this project demonstrates how to fetch and display Wikipedia data dynamically, offering a simple and engaging user experience.

---

## Features

- **Search Wikipedia Entries:** Enter a search term to fetch and display relevant Wikipedia articles.
- **Random Article Button:** Click to view a randomly selected Wikipedia article.
- **Custom Styling:** Personalize the app with your own styles and layouts.
- **Responsive UI:** Designed to work well on various devices.

---

## Technologies Used

- HTML5, CSS3, JavaScript
- Fetch API for HTTP requests
- (Optional) Any libraries or frameworks you prefer (e.g., Bootstrap, jQuery)

---

## User Stories

- **Search Functionality:** Users can input a search term and see relevant Wikipedia entries displayed.
- **Random Article:** Users can click a button to view a random Wikipedia article.
- **Dynamic Content:** Results update without full page reloads.
- **Personalization:** Feel free to style the app to match your personal aesthetic.

---

## How to Run the Project

1. **Clone or download** this repository.
2. Open `index.html` in your web browser.
3. Use the search box to find Wikipedia articles.
4. Click the "Random Article" button to explore random entries.
